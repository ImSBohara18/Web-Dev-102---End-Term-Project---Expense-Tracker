## Project Description
This is a fully functional, DOM-based Expense Tracker built using Vanilla JavaScript.  
The application allows users to add income and expense transactions, categorize expenses, calculate balance dynamically, and store data using LocalStorage.

## Problem Statement
Many students and individuals find it difficult to track their daily expenses and income.  
This project helps users understand their spending habits by providing a simple and interactive way to record and manage financial transactions directly in the browser.

## Features Implemented
- Add income transactions
- Add expense transactions with category selection
- Expense categories (Food, Travel, Shopping, Bills, Other)
- Dynamic DOM updates on user interaction
- Automatic calculation of income, expenses, and balance
- Data persistence using LocalStorage
- Delete transactions
- Input validation and error handling

## DOM Concepts Used
- DOM element selection (`getElementById`)
- Dynamic element creation (`createElement`)
- Updating DOM content dynamically
- Event handling (submit, click, change events)
- Conditional rendering using JavaScript
- Client-side state management
- LocalStorage usage

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## Steps to Run the Project
1. Clone or download the GitHub repository
2. Open the `index.html` file in any modern web browser
3. Add income and expense transactions
4. View live updates in balance and transaction history

## Known Limitations
- No backend or database integration
- No user authentication
- Data is stored only in the browser using LocalStorage
